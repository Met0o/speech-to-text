# Speech-to-Text Converter

A tool for converting audio files to text using Whisper AI model.

## Project Structure

```bash
├── app.exe
├── ffmpeg/
│   └── bin/
├── models/
│   └── large-v3-turbo.pt
|   └── large-v3.pt
├── whisper/
    └── assets/
        ├── gpt2.tiktoken
        ├── mel_filters.npz
        └── multilingual.tiktoken
```
